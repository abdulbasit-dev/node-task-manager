## Installation


- clone project

```bash
  git clone https://github.com/abdulbasit-dev/node-task-manager.git
  cd node-task-manager
```
- install dependencies

```bash
  npm install
```
- create .env file then get contnet from .env.example and put your value and credentials in there

- run app.js

```bash
  nodemon run dev
```
